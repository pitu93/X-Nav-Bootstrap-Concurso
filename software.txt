Dr. Scratch (http://drscratch.programamos.es/) es una plataforma web permite subir programas de Scratch, un lenguaje de programación visual orientado al aprendizaje de técnicas de programación para niños, para ser evaluados, de manera que el alumno recibirá realimentación sobre la calidad del programa, cómo mejorarlo y una evaluación global de sus capacidades de programación. La plataforma incluye técnicas de ludificación (gamification), de manera que el alumno puede ganar medallas y desarrollar sus habilidades de programación y alcanzar niveles de mayor complejidad y calidad. Dentro del proyecto se realizarán actividades específicamente orientadas al fomento de la programación para chicas, colectivos desfavorecidos y discapacitados.

LibreSoft Gymkhana (http://gymkhana.libresoft.es/) is an open source educational and geolocated game related to M-Learning & Tourism applications. It is based on the LibreGeoSocial mobile social network application, which includes a mobile augmented reality interface. You can organize and manage a mobile gymkana using Android smartphones (new generation gymkhanas) without great effort.
